Códigos de colores en bordes CSS

Color hexadecimal - #cc0000

Color HSL sigla del inglés Hue, Saturation, Lightness; en español, 'matiz, saturación, luminosidad' - hsl(0, 100%, 40%)

Color HWB es una representación de coordenadas cilíndricas de puntos en un modelo de color RGB , similar a HSL y HSV - hwb(0 0% 20%)

Color RGB (sigla del inglés red, green, blue; en español, ‘rojo, verde, azul’) - rgb(204, 0, 0)

Color RGBA son la siglas para red green blue alpha (en español: alfa verde azul rojo) - rgba(204, 0, 0, .3);
