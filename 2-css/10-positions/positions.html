Posicionamiento de tipo static

Esta es una etiqueta div con posición static: es la posición por defecto

Posicionamiento de tipo relative

Esta es una etiqueta div con posición relative: en este caso si nos afectan los atributos de top, bottom, left y right.

top: -10px; left: 50px

Posicionamiento de tipo fixed

Esta es una etiqueta div con posición fixed: en este caso nuestra posición nos permite mantener un elemento siempre fijo y sin que ocupe espacio con el resto de elementos. También respeta los atributos de top, bottom, right y left.

top: 50px; left: 40px

Posicionamiento de tipo absolute

Esta es una etiqueta div con posición absolute: en este caso nuestra posición nos permite mantener el elemento en una posición relativa al padre.

top: 30px; left: 50px; bottom: 0

Posicionamiento de tipo sticky

Esta es una etiqueta div con posición sticky: Cuándo salimos de la pantalla se comporta como una posición fixed, cuándo entramos en la pantalla se comporta por defecto. Si le ponemos un top, ese valor es en el que se va a cambiar la posición.

top: 0
